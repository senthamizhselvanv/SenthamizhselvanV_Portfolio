# Senthamizhselvan V - Portfolio Website

A premium, production-ready React portfolio website built with modern frontend technologies.

## 🚀 Features

- **Modern Design**: Premium UI inspired by Stripe, Linear, and Vercel
- **Dark/Light Mode**: Theme persistence with smooth transitions
- **Fully Responsive**: Mobile-first design approach
- **Animated UI**: Smooth animations using Framer Motion
- **SEO Optimized**: React Helmet for meta tags and Open Graph
- **Contact Form**: EmailJS integration for form submissions
- **Performance**: Lazy loading, code splitting, optimized assets

## 🛠️ Tech Stack

- React 18
- React Router
- Framer Motion
- Bootstrap 5
- SCSS
- React Icons
- React Type Animation
- React Helmet Async
- EmailJS
- Vite
